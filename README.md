# RubikTeacher

Rubik Teacher is an application to solve a Rubik's Cube and learn how to solve it by yourself

It implements several solving methods:
- Kociemba: optimally computed, not teachable for humans (~20-25 moves)
- Fridrich: Advanced method used in speedcubing (~60 moves)
- Layer by Layer: Beginner method (~ 100 moves)

![preview](https://sebastienlabonne.dev/res/rubikTeacher/rt_learn.png)


## Compilation

Compile with Qt6, MinGW using CMake file
